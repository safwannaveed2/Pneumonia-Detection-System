A deep learning-powered web application to detect **Pneumonia** from chest X-ray images. Built using **TensorFlow**, **OpenCV**, and **Streamlit**, this system provides a fast and accurate way to screen for pneumonia in medical images.

---

## Features

- Upload chest X-ray images in `jpg`, `jpeg`, or `png` format.
- Real-time pneumonia detection with confidence-based predictions.
- Clean and interactive UI powered by Streamlit.
- Pre-trained deep learning model for accurate classification.
